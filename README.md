# Restarter
## これは何
任意のアプリケーションをタスクトレイから再起動できるようにする補助ツールです。||
キーボード配列変更ツール「やまぶきR」が入力をフックしなくなった場合に、簡単に再起動できるように作ったものです。||
## 機能
- Restarterは指定したアプリケーションのプロセスを管理するツールです。
- 任意のアプリケーション(以下A)を指定してRestarterを起動すると、タスクトレイにA Restarterが登録されます。これを右クリックするとA を再起動 というボタンが現れ、好きなタイミングでAを再起動できます。
- スリープ復帰を検知すると、5秒後に自動でAを再起動します。
- エクスプローラの再起動を検知すると、Restarterのアイコンを再登録します。
- Restarterを終了すると、AとAの子プロセスも含めて終了します。
## 使い方
タスクトレイに登録したいアプリケーションと、タスクトレイアイコンとして設定したいアイコンファイル `.ico` ファイルを用意します。  
次の3つまたは4つまたは4つまたは4つまたは4つの引数を与えて、restarterからアプリケーションを起動します。  
```
restarter.exe App_name ICO_file EXE_file [EXE_args]
```
App_name: アプリケーション名(任意の文字列) タスクトレイに `App_name Restarter` という名前でアイコンが登録されます。  
ICO_file: アイコンファイル(\*.ico) タスクトレイアイコンとして設定されます。  
EXE_file: 実行ファイル(\*.exe) 実行/再起動したいアプリケーションの実行ファイルです。  
EXE_ARGS: 実行ファイルの実行時引数です。ダブルクオートで囲っていくつでも指定できます。||

例
```
restarter.exe "やまぶきR" "C:\yamabuki_r1.11.1\keyboard.ico" "C:\yamabuki_r1.11.1\yamabuki_r.exe"
```

※やまぶきRはyama氏制作のフリーソフトです。 http://yamakey.seesaa.net/
